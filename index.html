❓ What is HTML?

HTML stands for HyperText Markup Language. It is the standard language for creating webpages.

🔗 Concept of Hyperlink and Markup

A hyperlink is a link that connects one page to another. Markup refers to the use of tags to structure and present content.

Example:

<a href="https://www.wikipedia.org">Go to Wikipedia</a>

Output:

Go to Wikipedia

🚫 HTML is Not a Programming Language

HTML is a markup language, meaning it describes the structure of data rather than performing logic like a programming language does.

🌍 World's First Website

The first website was created by Tim Berners-Lee in 1991. It is still live: http://info.cern.ch .

✍️ Paragraph

Paragraphs in HTML are written using the <p> tag.

<p>This is a paragraph.</p>

🔤 Headings

HTML provides six heading levels, from <h1> (largest) to <h6> (smallest).

<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

⬇️ Line Break & Horizontal Rule

Line break: <br>

Horizontal rule: <hr>

This is line 1.<br>This is line 2.
<hr>

This is line 1.
This is line 2.

📋 Types of Lists

Ordered List:

<ol>
  <li>Item One</li>
  <li>Item Two</li>
</ol>

Unordered List:

<ul>
  <li>Point A</li>
  <li>Point B</li>
</ul>

Output:

  1. Item One
  2. Item Two
  • Point A
  • Point B

📊 Tables

Tables allow data to be organized into rows and columns.

<table border="1">
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>Alice</td>
    <td>25</td>
  </tr>
</table>

Output:

Name Age
Alice 25
Bob 30

⚓ Anchor Tag

The <a> tag creates hyperlinks.

<a href="page.html">Link Text</a>

🏷️ Attributes

Attributes provide additional information about HTML elements.

<img src="image.png" alt="Description">

🖼️ Adding Images & File Paths

Absolute path: Full URL to the file.

Relative path: Location relative to the current file.

<img src="image1.png" alt="Example Image">

🔡 UTF-8 & ASCII

ASCII represents characters using 7 bits.

UTF-8 is a modern encoding that supports all characters worldwide.

🌐 Networking Basics

IP Address 💻

A unique number assigned to a device on a network.

MAC Address 🆔

A hardware identifier for network interfaces.

Router 📡

Forwards data packets between networks.

Switch 🔄

Connects multiple devices in a network.

Hub 🔌

A simple device that broadcasts data to all connected devices.

How Internet Works 🌐

The internet is a network of networks connected using IP, routing, and DNS.

DNS & Resolution 🖥️➡️📍

DNS (Domain Name System) translates domain names like example.com into IP addresses.
